# 📚 AI-Powered Research Paper Summarization and Insight Generation Agent

An intelligent research assistant that helps researchers, students, and academics efficiently analyze research papers using AI.

## 🎯 Features

- **📝 Automatic Summarization** - Generate concise, structured summaries of research papers
- **🔍 Information Extraction** - Extract key components (problem, methodology, results, limitations)
- **📊 Paper Comparison** - Compare multiple research papers side-by-side
- **🎯 Research Gap Identification** - Identify gaps and suggest future research directions
- **❓ Question Answering** - Answer specific questions about research papers
- **📄 PDF Support** - Upload PDF files directly
- **🎨 Web Interface** - User-friendly Streamlit interface
